Contient les outils suivants :
- Laravel 8
- VueJS 2.7.8
- Vuetify 2.6.9
- Vue Router 3.5.4
- VueX 3.6.2
- Axios

Installation :
- `npm install`
- `composer install`
- `npm run watch`

Nécessite composer, PHP 8

Il faut ensuite créer un fichier .env à la racine du projet.

`php artisan key:generate` si besoin.
